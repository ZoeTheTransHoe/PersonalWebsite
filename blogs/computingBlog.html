Linux

I use Linux on my Desktops and Minecraft Server. I have been The Linux Distros I reccomend the most are:

Fedora Linux.

My preferred Linux Distro for the Desktop, reliable yet always on the leading edge, while commited to open-source.
I am also an active member of the Fedora Community.

Debian Linux.

The Linux Distro I use for my Minecraft Server. It focuses on being extremely stable for long periods of time, while also being commited to open-source.
It also incredible support for more neiche hardware architectures, like the Raspberry Pi.

Arch Linux.

The Linux Distro I use for containers like Distrobox. It focuses on being extremely light-weight and being extensibly tweakable.
The remarkbly helpful Arch Wiki with tips and instructions for all sorts of Linux shenanigans, and the AUR for installing all kinds of software.

NixOS.

An awesome distro for Linux Developement across many machines, using Nix's declerative package manager and configuration files, with extremely well stocked repos, and a rock-solid base.
You can check out my NixOS configuration files here.

Gentoo Linux.

A fun and unique Linux Distro for Tinkering. While not for the light-hearted at all, its source-based package management allows for absurd levels of hardware architecture support.
It gives the user complete control of how every application is built, including removing parts of the program they don't need using special USE flags.

VinallaOS.

My favourite up-and-coming Linux Distro, its an immutable distro with many useful utilities, including the apx package manager for managing containers of many types of Linux Distros (including some of the others mentioned!)
It's an interesting look at the future of Linux Distros, and allowing for greater control and ease of use while making Linux harder to break XD.
